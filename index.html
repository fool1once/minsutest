About This Game

Created by: Jhon Benedict Zulueta


This quiz game was created as part of a learning project to improve both programming and understanding of the Filipino language. The goal of this game is not only to test knowledge but also to make learning interactive and enjoyable.


I built this system using HTML, CSS, and JavaScript to practice front-end development, including handling user interaction, timers, scoring systems, and local storage for leaderboard tracking. This project reflects my passion for technology and my goal of becoming better in software development and cybersecurity in the future.
